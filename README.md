# Crazy Random Fun Zone

A silly, interactive website for kids with sounds, dancing Pokemon, and fun facts!

## Features

- **Silly Sound Machine** - Make random boing, splat, and honk sounds
- **Silly Face Maker** - Generate wacky emoji face combinations
- **Pikachu Twerk Party** - Watch Pikachu dance!
- **Dance Battle Arena** - Vote for your favorite Pokemon dancer
- **Random Fun Facts** - Learn silly facts about animals
- **Crazy Mode** - Activate for confetti, screen shake, and chaos!

## Tech Stack

- **Vite** - Build tool
- **React 19** - UI library
- **TypeScript** - Type safety
- **Tailwind CSS v4** - Styling
- **Web Audio API** - Sound effects

## Getting Started

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev

# Build for production
pnpm build
```

## Design

Cartoon Explosion Lab aesthetic inspired by Nickelodeon cartoons:
- Bright saturated colors (hot pink, lime green, electric blue)
- Thick black outlines
- Bouncy animations
- Playful typography (Fredoka, Baloo 2, Nunito)
